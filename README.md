# Quiz Optimization System

A full-stack web application that helps users maximize their quiz scores within a specific time limit. It uses a **Dynamic Programming (Knapsack Algorithm)** approach to determine the optimal set of questions to answer.

## üöÄ Features

* **Quiz Dashboard:** Browse available technical and general knowledge quizzes.
* **Smart Optimization:** Input a time limit (e.g., 5 minutes), and the system calculates the mathematically perfect combination of questions to answer for the highest possible score.
* **Attempt History:** Tracks selected strategies and scores in a database.
* **Modern UI:** Clean, minimalist design using Tailwind CSS.
